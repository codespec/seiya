# seiya

Restaurant management tool


## Development

```sh
$ yarn install
# Run development mode
$ yarn start
```

**Note**: When you add a package, please add with `yarn` to update `yarn.lock` file.

## Run Production

```sh
yarn global add serve
serve -s build
```
